# Configuration

For the configuration, `config.js` and replace the :

```js
module.exports = {
  token: "",
  botID: "",
  ownerID: "",
  privateServerID: "",
  embedColor: "",
  pingcustom: "",
  embedinvisiblecolor: "#36393F",
};
```

# Commands

**Installation**

> `npm install`

**Start the bot**

> `node index.js`
